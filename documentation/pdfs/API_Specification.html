API Specification

Resource: files

Method Endpoint Description
POST /api/files/upload Upload a new image file (max 10MB)
GET /api/files/{fileName} Retrieve an image file by filename
DELETE /api/files/{fileName} Delete an image file

Resource: items

Method Endpoint Description
GET /items Retrieve all items with filters/search
GET /items/:id Retrieve a specific item by ID
PUT /items/:id Edit item details (e.g. mark as claimed)
DELETE /items/:id Delete an item

Resource: reports

Method Endpoint Description
GET /reports/:id Retrieve a specific report by ID
POST /reports Create a new report (and related item/location)
PUT /reports/:id Edit report details
DELETE /reports/:id Delete a report

Resource: locations

Method Endpoint Description
GET /locations Retrieve all locations
GET /locations/:id/items Retrieve items at a specific location
POST /locations Add a new location
PUT /locations/:id Edit location details
DELETE /locations/:id Delete a location

Resource: users

Method Endpoint Description
POST /users Create user
PUT /users/:id Edit user details
